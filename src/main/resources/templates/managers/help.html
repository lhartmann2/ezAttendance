  • Getting Started
There are 4 steps that need to be done in order to ensure success in using this app:
  1. Create Parents / Guardians from the 'Parents' tab.
  2. Create Students from the 'Students' tab, and associate at least one parent / guardian.
  3. Create Teachers from the 'Teachers' tab
  4. Create Classes from the 'Classes' tab, and associate a teacher and one or more students.
In short, students need to be attached to at least one parent / guardian, and classes need to be attached to a teacher and one or more students.
  • Taking Attendance
Navigate to the attendance menu from the 'Attendance' Tab.
By default, the page will show classes occurring today by all teachers.
You can filter classes by time frame (today, this week, this month, or a specified date range) in addition to filtering by a particular teacher.
To take attendance for a particular class session, press 'Take Attendance' on the class.
For speed and ease of use, all students are marked as 'present' by default.
To mark absent, simply click or tap the 'present' slider.
This will reveal another slider, used to indicate if the student's absence is excused or not.
Below the sliders is a text box that can be used to enter details, such as a reason for being absent (sick, on vacation, etc.) or other information teachers may want to see at the beginning of class.
After pressing 'Submit', the record will be saved and the class session's attendance will be marked as 'Taken'.
To edit the attendance record, simply press 'Take Attendance' on the class again, or by clicking 'View Record' and then clicking 'Edit'.
  • Attendance Reports
There are two ways to generate attendance reports:
  1. For One Class Session:
    In the Attendance menu, find the class session you want to see a report on. Filter by date and teacher as needed.
    Press 'View Record' to see the list of students and their attendance for that particular class session.
    By pressing 'Report', you can choose to export the attendance report as a text file or CSV (Excel) document.
  2. By Student:
    In the Attendance menu, press the 'Report' button.
    From this page, select the student you want an attendance report for.
    Next, you will be asked which of the student's classes (or all classes) you want a report on.
    (This page can also be accessed from the Student menu and pressing 'Report' on a student).
    Finally, you will be asked if you want to focus on a particular range of time or if you want the student's entire history.
    You will then be presented with a list of the student's attendances.
    Using the buttons at the top of the page, you can export the attendance report as a text file or CSV (Excel) document.
  • Teachers
Navigate to the Teachers tab in the top menu.
A list of teachers can be seen. They can be sorted by first or last name.
Pressing 'Classes' will show a list of classes taught by that teacher.
  • Adding Teachers
    To add a teacher, press the 'Add Teacher' button at the top of the page.
    You will be presented with a form where you can fill out the teacher's information.
    A phone number is required but an email address is optional.
  • Editing and Deleting Teachers
    If you need to change a teacher's information, locate their name on the teacher list and press 'Edit'.
    From the edit page, a teacher can be deleted if needed.
Note: There must be at least one teacher before a class can be created.
  • Parents
Navigate to the Parents tab in the top menu.
A list of parents / guardians can be seen. They can be sorted by first or last name.
To see a parent / guardian's students, press 'Students' on a particular entity. Press 'Details' to see more information such as contact, address, notes, etc.
  • Adding Parents
    To add a parent or guardian, press the 'Add Parent' button at the top of the page.
    You will be presented with a form where you can fill out the parent or guardian's information.
    Everything is optional except for a first and last name and a phone number.
    Anything filled out here can be seen when clicking on a parent / guardian's details anywhere in the app.
  • Editing and Deleting Parents
    To add or edit information to a parent or guardian, find their name in the parents list and press 'Edit'.
    From the edit page, a parent / guardian can be deleted with the 'Delete' button if needed.
    If a parent is deleted and is attached as the primary parent to a student, the student's primary parent
    slot will be blank and one will need to be manually added.
    (See the 'Students' topic on how to attach parents to a student).
Note: There must be at least one parent before a student can be created, as every student must be attached to at least one parent.
  • Students
Navigate to the Students tab in the top menu.
A list of students can be seen. They can be sorted by first name, last name or by age (ascending or descending).
Pressing 'Classes' for a particular student will show a list of classes they are currently enrolled in.
Pressing 'Details' for a particular student will show additional information, such as their parents / guardians, allergies, preferred hospital, etc.
Pressing 'Report' for a particular student will start the process of generating an attendance report.
(See the 'Taking Attendance' topic).
  • Adding Students
    To add a student, press the 'Add Student' button at the top of the page.
    Important: Before continuing, ensure a parent or guardian is created (See the 'Parents' topic).
    You will be presented with a form where you can fill out the student's information.
    Everything is optional except for a first and last name, birthday, and a primary parent or guardian.
    To attach a parent or guardian, press the dropdown box and select the previously created parent.
    Optionally a second parent or guardian can be attached.
    Information entered here can be seen when clicking on a student's details anywhere in the app.
  • Editing and Deleting Students
    To add or edit information to a student, find their name in the students list and press 'Edit'.
    Note: If you previously deleted a parent that was attached to a student as their primary parent,
    the primary parent / guardian dropdown box will be blank and a new parent / guardian must be attached.
    From the edit page, a student can deleted with the 'Delete' button if needed.
    If a student is deleted, they will be removed from all enrolled classes and any attendance data will be removed.
  • Classes
Navigate to the Classes tab in the top menu.
A list of classes can be seen. They can be sorted by current, past (classes that have ended),
future (classes that haven't started yet) and all (current, past and future).
Pressing 'Students' for a particular class will show the students enrolled.
From here you can see each student's details (contact, parents, allergies, etc.),
the classes they are enrolled in, generate an attendance report, or directly edit them.
Pressing 'Details' for a particular class will show more information such as the dates and times of the class.
  • Adding Classes
    To add a class, press the 'Add Class' button at the top of the page.
    Important: Before continuing, ensure that at least one teacher and one student has been created
    (See the 'Teachers' and 'Students' topics).
    You will be presented with a form where you can fill out the classes details.
    You can add one or more students using the student dropdown search box.
    Information entered here can be seen when clicking on a class' details anywhere in the app.
  • Editing and Deleting Classes
    To edit a particular class, find it in the Class list and press 'Edit'.
    WARNING: Making changes to the start date, end date or day of the week of a class
    will result in all previously taken attendance data being reset!
    From the edit page, a class can be deleted with the 'Delete' button.
    All attendance data for a deleted class will also be deleted.
  • Changing Password
At the bottom of any page, press the 'Account' link.
Press the 'Change Password' button and enter a password that is at least 8 characters long.
Use a strong password that is unlikely to be guessed to ensure the security of the app and its data.
  • Deleting Account
At the bottom of any page, press the 'Account' link.
Press the 'Delete Account' button. You will be asked to confirm your choice.
Note: Deleting user accounts does not affect any other data.
  • Admins: Creating Accounts
At the bottom of any page, press the 'Admin' link.
Press the 'Create User Account' button.
Enter a username and password that is at least 8 characters long.
Use a strong password that is unlikely to be guessed to ensure the security of the app and its data.
You can set whether the new account is an admin or not. This cannot be changed later, so carefully
consider who is allowed to have admin privileges.
  • Admins: Changing Passwords
At the bottom of any page, press the 'Admin' link.
Press the 'Change Passwords' button.
Select the username you wish to change the password for and enter a new password that is at least 8 characters long.
Use a strong password that is unlikely to be guessed to ensure the security of the app and its data.
  • Admins: Unlocking Accounts
For security purposed, users who try to login with an invalid password 5 times will have their account locked for 24 hours.
Admins can manually unlock accounts if needed.
To unlock an account, at the bottom of any page, press the 'Admin' link.
Press the 'Unlock Accounts' button.
Select the account you wish to unlock and reset failed login attempts for.
You may choose to reset the user's password at this time, too.
  • Admins: Deleting Users
Admins can delete user accounts and other admins.
To ensure there is always at least one admin on the system, admins cannot delete themselves.
To delete an account, at the bottom of any page, press the 'Admin' link.
Press the 'Delete Users' button.
Select the user you wish to delete and press the 'Delete' button.
Exercise caution in this menu as users cannot be undeleted.
Note: Deleting users does not affect any other data.
  • Admins: Upcoming Events Manager
Upcoming events are dates shown on the homepage to all users.
To manage upcoming events, at the bottom of any page, press the 'Admin' link.
Press the 'Upcoming Events Manager' button.
From here you will see a list of events, with links on each to edit and delete them.
Press the 'Create Event' to add an entry.
Events with dates in the past will not appear on the homepage, so deleting past events is not necessary.
  • Admins: Contact Lists
Contact Lists are an easy way to get contact info for parents, teachers and students all at once.
To see the contact lists, at the bottom of any page, press the 'Admin' link.
Press the 'Contact Lists' button.
Select if you want the contact lists for parents, teachers or students.
Parents' contact lists can be sorted by class. This is useful in the situation of a class being cancelled and all affected parents need to be contacted.
From any contact list page, the information can be exported using the buttons at the top to a text file or CSV (Excel) document.
